# 俺の嫁がこんなにも可愛い
## 概要
俺の嫁、つまり自分が好きなキャラ、愛しているキャラについてどこが好きなのか、こんなところがいいなどなどとにかく好きなキャラについてひたすら投稿するだけのSNS

## メンバー
梅﨑貴史：端末・サーバー
